Using

Add css

<link rel="stylesheet" type="text/css"href="bootstrap.dropdownform.css">

Add script

<script src="bootstrap.dropdownform.js"></script>

Add html

<div class="btn-group">
    <button data-toggle="dropdownform" class="btn dropdownform-toggle">Dropdown Form with
    Dropdowns</button>

    <ul class="dropdown-form" style="width:600px;">
      <li>
        <form style="margin:15px;">
          <div class="row-fluid" style="margin-top:10px;">
            <div class="span2">
              Action type:
            </div>

            <div class="span10">
              <div class="btn-group">
                <a class="btn" href="#">Single Family</a> <a class="btn dropdown-toggle"
                data-toggle="dropdown" href="#"></a>

                <ul class="dropdown-menu">
                  <li><a href="#">Single Family</a></li>

                  <li><a href="#">Manufactured</a></li>
                </ul>
              </div>
            </div>
          </div>
        </form>
      </li>
    </ul>
  </div>

Build

Install grunt

$ git clone git@github.com:olegsmith/bootstrap.dropdownform.git
$ cd bootstrap.dropdownform
$ npm install -g grunt-cli (if grunt-cli is not installed)
$ npm install grunt@0.4
$ npm install grunt-contrib-uglify

Build

$ grunt
